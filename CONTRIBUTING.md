# Contribution Guidelines

Thank you for considering contributing to the WanderAI project! Here are some guidelines to help you make your contributions as smooth as possible:

## How to Fork
1. Click on the **Fork** button at the top right of the repository page on GitHub.
2. This creates a copy of the repository under your account.

## Create Feature Branches
1. Create a new branch for your feature:  
   `git checkout -b feature/my-feature`

## Make Changes
1. Make your changes in the feature branch.

## Test
1. Run all the tests to ensure that your changes don't break anything:  
   `npm test` (or the relevant command).

## Commit
1. Add your changes:  
   `git add .`
2. Commit your changes with a meaningful message:  
   `git commit -m "Add my feature"`

## Push
1. Push your changes to your feature branch:  
   `git push origin feature/my-feature`

## Create a Pull Request
1. Navigate to the original repository where you want to propose your changes.
2. Click on **New Pull Request**.
3. Select your feature branch and fill out the relevant information.
4. Click **Create Pull Request**.

## Code Style Guidelines
- Follow the existing code style in the repository for consistency.
- Use 2 spaces for indentation.
- Prefer using `const` and `let` instead of `var`.

## Issue Reporting
- To report an issue, navigate to the **Issues** tab and click on **New Issue**.
- Provide a clear description and steps to reproduce the issue.

## Pull Request Process
- Your pull request will be reviewed by the maintainers of the project. Be open to feedback and make necessary adjustments.

## Commit Message Guidelines
- Use the present tense ("Add feature" instead of "Added feature").
- Be concise but descriptive.

## Development Setup Instructions
1. Clone the repository:  
   `git clone https://github.com/anchi7777/wanderai.git`
2. Navigate into the directory:  
   `cd wanderai`
3. Install dependencies:  
   `npm install`

## Contact Information
For questions or support, please reach out to anchi7777 via GitHub or any other preferred method of communication.
